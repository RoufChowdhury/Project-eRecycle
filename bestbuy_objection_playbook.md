# Best Buy Objection Playbook

**Mode:** Calm, factual, non-defensive  
**Rule:** If evidence is incomplete, label as `HYPOTHESIS` and convert into pilot test criteria.
**Evidence tags:** Use the shared registry in `bestbuy_talk_track.md`.

`deep_answer_90s`: "Your current ecosystem remains intact. The pilot is scoped to one store and specific categories so you can evaluate additional control and reporting without operational disruption. If value is not proven, the pilot stops cleanly."
