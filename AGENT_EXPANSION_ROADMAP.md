```bash
# ─── CODING ──────────────────────────────────────

# AGT5-CL: Cline (VS Code)
# Install: VS Code Extensions → search "Cline" → Install

# ─── RESEARCH ────────────────────────────────────

# AGT7-PX: Perplexity Pro

# AGT8-AT: ChatGPT Deep Research
# Use via browser: chatgpt.com → Deep Research mode

# AGT9-EL: Elicit + SciSpace

# ─── AUTOMATION ──────────────────────────────────
```
