Sorting, Testing & Classification

Every item inspected, tested, graded, and documented
üá®üá¶ Canada ‚ñ∂
üîç 6-Point Inspection Protocol
  • Power-On Test ‚Äî Device must boot to OS / power indicator
  • Core Functionality ‚Äî Screen, keyboard, ports, WiFi, motor (appliances)
  • Safety Inspection ‚Äî No swollen/leaking batteries, no exposed wires, no cracks
  • Cosmetic Assessment ‚Äî Physical condition scoring (1-5 scale)
  • Photo Documentation ‚Äî Timestamped photos of each item, front/back/serial
  • Identity Logging ‚Äî Serial number, make, model, year, category recorded
‚üê CLASSIFICATION DECISION GATE

Based on the 6-point inspection, each item is classified into one of four grades. This classification determines the item's entire downstream path.

A Reusable
  • Fully functional
  • Good cosmetic condition
  • Less than 5 years old
  • ‚Üí Export for direct resale
B Repairable
  • Minor issues only
  • Fixable with <$30 in parts
  • Core functions intact
  • ‚Üí Export for refurbishment
C Parts Harvest
  • Non-functional device
  • Contains valuable components
  • No hazardous materials
  • ‚Üí Export for disassembly
X Hazardous
  • Damaged/swollen batteries
  • ODS refrigerants (AC/fridge)
  • Leaking, unsafe materials
  • üö´ NEVER EXPORTED
üõ°Ô∏è
Grade X Diversion Rule: All Grade X items are immediately diverted to a certified Canadian recycler (R2 or e-Stewards certified). They never enter the export pipeline. This is a legal and ethical non-negotiable under Basel Convention and NESREA regulations.

Basel Convention 2025 Amendment: As of January 2025, ALL e-waste exports (including non-hazardous) require Prior Informed Consent (PIC) from both exporting and importing countries. This permit must be obtained BEFORE any shipment leaves Canada. Filing timeline: 4-6 weeks before planned export date.

‚üê NESREA INSPECTION GATE

If ANY item fails inspection (non-functional, waste characteristics, missing docs), the entire container may be refused entry, fined (‚Ç¶5,000,000+), or returned to origin. This is why pre-shipment testing is critical.
